sort1 uses: Bubble sort

How do you know?: This sort has the best times for already sorted data.
    It's quite slow when data is large and speed when data is small.

sort2 uses: Merge sort

How do you know?: This is the fastest sort for large data and times are very similar
     no matter data is sorted random or reversed.

sort3 uses: Selection sort

How do you know?: It has nearly the same time for the same amount of data.
